# Study Routines

These routines help me stay consistent throughout the semester.

---

## Daily Plan
- Check the dashboard in `web-interface/index.md`  
- Review notes from the previous day  
- Work on one assignment or lab, even if only for 20–30 minutes  
- Keep a running list of questions to ask instructors  
- End the day with a quick summary of what I learned  

---

## Weekly Plan
- Update each course folder with notes or progress  
- Convert rough notes into clean versions  
- Review upcoming deadlines  
- Refresh concepts using flashcards or short quizzes  
- Spend time on cybersecurity hands-on practice  

---

## Course Workflow
1. **Read or watch lecture material**  
2. **Take raw notes** (messy is fine)  
3. **Clean notes** in the `notes/` folder  
4. **Use AI only to clarify or confirm**  
5. **Do assignments and labs myself**  
6. **Reflect** on what I learned or struggled with  

---

## Semester Goals
- Stay ahead in cyber courses  
- Strengthen math fundamentals early  
- Build consistent habits so upper-level courses are easier  
- Capture everything in the repo so I can look back and build on it
